# Linux Image Rendering Fix — исправляет битые изображения на Linux

**Репозиторий:** [GitHub](https://github.com/Evgene-Kopylov/linux-image-rendering-fix)

**Установка:** Community Plugins → Browse → **Linux Image Rendering Fix**

---

## Проблема

Пользователи Linux (Ubuntu, Pop!_OS, Linux Mint, Fedora и другие) мучаются с тем, что Obsidian не отображает локальные изображения уже **три года** (с версии v1.0.0). Вместо картинки — битая иконка или пустое место. На уровне приложения проблема до сих пор не исправлена.

**Причины:**

1. **Сломанный протокол `app://`** — Electron/Chromium на Linux не может разрешить пути, содержащие не-ASCII символы: эмодзи, диакритику, кириллицу, CJK и другие.
2. **Отсутствие кодеков изображений** — некоторые сборки Electron для Linux (особенно Snap и Flatpak) поставляются без кодеков PNG, JPEG, WebP, что приводит к status 0 или ошибкам декодирования.

Затронутые пакеты: **Snap**, **Flatpak**, **DEB**. AppImage чуть стабильнее, но не застрахован. ARM-системы тоже под ударом.

Это upstream-баг Electron/Chromium. Команда Obsidian не может исправить его напрямую.

## Решение

Плагин перехватывает рендеринг Markdown через `MarkdownPostProcessor`, находит каждый элемент `<img>`, читает файл изображения напрямую с диска с помощью `vault.adapter.readBinary()` и заменяет `src` на Blob URL (`blob:...`). Это полностью обходит сломанный протокол `app://`.

**Проверено на:** Ubuntu 24.04 (DEB-пакет).

Другие дистрибутивы (Pop!_OS, Linux Mint, Fedora и т.д.) и форматы пакетов (Snap, Flatpak, AppImage), вероятно, тоже подвержены проблеме — первопричина в upstream-баге Electron/Chromium, а не в конкретном дистрибутиве.

## Возможности

- **Автоматическая обработка** — изображения подменяются при открытии страницы, переключении вкладок и изменении файла
- **Широкая поддержка форматов:** PNG, JPEG, GIF, WebP, SVG, BMP, ICO
- **Команда Reprocess Images** — ручное обновление изображений в текущем представлении
- **Безопасность по дизайну:** работает только с файлами внутри хранилища, не делает сетевых запросов
- **Нулевая настройка:** установил и забыл

## Ограничения

- **Только десктоп (Linux).** `isDesktopOnly: true`
